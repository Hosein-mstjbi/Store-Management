# سیستم مدیریت فروشگاه مواد غذایی

این پروژه یک برنامه کنسولی برای مدیریت فروشگاه مواد غذایی است که قابلیت‌های زیر را فراهم می‌کند:

## قابلیت‌ها

* افزودن محصول جدید
* خرید محصول (افزایش موجودی انبار)
* فروش محصول (کاهش موجودی)
* مشاهده فهرست کامل محصولات
* مشاهده موجودی هر محصول

## تکنولوژی‌ها

* Java
* JDBC
* PostgreSQL

## هدف پروژه

پیاده‌سازی یک سیستم ساده برای مدیریت انبار و تراکنش‌های فروشگاه با استفاده از اصول پایه‌ای جاوا و ارتباط با دیتابیس.
